## **Featured: Apache NiFi 2.9.0 Released**

**Apache NiFi 2.9.0** is here with over 150 issues resolved! This is a significant release that adds initial support for **Connectors** - a new NiFi concept - plus **Google Cloud Storage support for Iceberg** and a refactored ConsumeKinesis processor. A key part of the SNACK-AI stack continues to evolve.

**Key Highlights:**
- Initial support for Connectors
- Google Cloud Storage support for Iceberg
- Refactored ConsumeKinesis to avoid KCL dependency
- Bug fix for ConsumeKafka duplicate message processing during rebalance

**Release Notes:** https://cwiki.apache.org/confluence/display/NIFI/Release+Notes#ReleaseNotes-Version2.9.0

## **Snowflake Platform Updates**

### Budgets for AI Features (GA - Apr 10)

Use **custom budgets** to track and control credit consumption for AI features - AI Functions, Cortex Code, Cortex Agents, and Snowflake Intelligence - broken down by team or cost center. Tag users to a business unit and the budget tracks only their credits.

**Read more:** https://docs.snowflake.com/en/release-notes/2026/other/2026-04-10-budgets-ai-features-ga

### AI_SERVICES Billing Breakout (Apr 6)

More granular billing service types for AI Credits. Cortex Agents, Cortex Code CLI, Cortex Code Snowsight, and Snowflake Intelligence are now broken out of `AI_SERVICES` as separate `SERVICE_TYPE` values in metering views.

**Read more:** https://docs.snowflake.com/en/release-notes/2026/other/2026-04-06-ai-services-billing-breakout

### Medical and Health Data Classifiers (GA - Apr 3)

### Performance Explorer Granular Access Control (Apr 2)
