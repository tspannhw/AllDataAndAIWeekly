## **Featured: Catalog-Linked Databases with Cortex Code CLI**

The Snowflake engineering team published a comprehensive guide on using **Catalog-Linked Databases** with the Cortex Code CLI for Apache Iceberg workflows. This enables seamless integration between external Iceberg catalogs and Snowflake's AI capabilities.

**Key Capabilities:**
- Connect to external Iceberg catalogs (AWS Glue, Polaris, etc.)
- Query Iceberg tables directly from Cortex Code
- Build AI pipelines on open table formats
- Maintain full interoperability with other query engines

**Read more:** https://www.snowflake.com/en/engineering-blog/catalog-linked-databases-cortex-code-cli-iceberg/
